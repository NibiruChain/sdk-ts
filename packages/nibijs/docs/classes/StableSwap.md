### y

▸ **y**(`fromIndex`, `toIndex`, `x`): `BigNumber`

y()

Calculate x[j] if one makes x[i] = x

Done by solving quadratic equation iteratively.
x*1\*\*2 + x1 * (sum' - (A*n\*\*n - 1) * D / (A _ n**n)) = D ** (n+1)/(n \*\* (2 _ n) \_ prod' \* A)
x_1\*\*2 + b\*x_1 = c

x_1 = (x_1\**2 + c) / (2*x_1 + b)
